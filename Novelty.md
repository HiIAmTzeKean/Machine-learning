# Novelty
Novelty is computed as the minus logarithm of (number of interactions with item / total number of interactions).

The novelty of an item can be defined relative to a set of observed events on the set of all items. These can be events of user choice (item â€œis pickedâ€ by a random user) or user discovery (item â€œis knownâ€ to a random user). The above definition of novelty reflects a factor of item popularity. High novelty values correspond to long-tail items in the density function, that few users have interacted with and low novelty values correspond to popular head items.

Reference: P. Castells, S. Vargas, and J. Wang, Novelty and diversity metrics for recommender systems: choice, discovery and relevance, ECIR 2011
