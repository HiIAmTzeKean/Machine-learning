# Logistic function
## Single [[Predictor]]
$$P(X) = \frac{e^{\beta_0+\beta_1X}}{1+e^{\beta_0+\beta_1X}}$$
## Multiple predictor
$$P(X) = \frac{e^{\beta_0 + \beta_1X1+...}}{1-e^{\beta_0 + \beta_1X1+...}}$$
